I2B (ISO 2 Boot) v0.1.0-alpha.1

A simple app to make bootable USBs.

STATUS:
This small project is in alpha. It works, but it's expected to get bugs and is in active development.
Features will change soon and bugs will be fixed aswell soon.

REQUIREMENTS:
Windows 10 or later.
Administrator privileges (good for you if you are already an admin.)

HOW TO USE:
1.Launch the app (duh.)
2.Select the ISO you want to use.
3.Select the target USB.
4.Make sure the correct USB is selected.
5.Press write and wait.

BUILDING FROM SOURCE:

Note: This is a project in Visual Studio. Make sure you have Visual Studio installed.

1.Open the solution (in .sln).
2.Build what you want.
3.Run it from the 'bin' folder.
